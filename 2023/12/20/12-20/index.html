Clash和订阅

Clash简介

​ Clash 是一个使用 Go 语言编写，基于规则的跨平台代理软件核心程序。

​ Clash for Windows 是目前在 Windows 上最好用的图形化分支。通过 Clash API 来配置和控制 Clash 核心程序，降低了 Clash 的使用成本，小白也能轻松入门使用。

下载地址

​ https://doc.miyun.app/apps/clashforwindows/Clash.for.Windows.Setup.0.20.39.exe

配置

​ 1、下载后安装，无脑下一步即可，也可以设置安装路径,最后桌面生成快捷方式；打开桌面的Clash for windows。

​ 2、购买相关的订阅然后复制订阅链接，这里提供一个一元机场的网址(附带邀请码) https://xn--4gq62f.com/#/register?code=iL2EQ3wT。

​ 3、选择购买订阅，有三种购买的计划，上面也有详细的说明。

​ 4、购买以后复制我们订阅链接。

​ 5、通过订阅链接下载配置文件，在(Profiles)配置文件一栏里面，粘贴你购买订阅的链接URl，然后点击download(下载)订阅文件,最后会新的配置文件(一元机场)，点击选择它就行。

​ 6、切换到 proxies（代理）一栏，会看到很多节点，单击无线图标可以查看延迟等。选择你的节点，用鼠标点下即可。其他不懂的不要动。

代理模式

​ 电脑任务栏下方的clash右键点击有很多选项

  • 全局（Global）：所有请求直接发往代理服务器，上面开启系统代理需要选择这个模式，如果不选择这个模式，比如选择Rule模式，许多操作需要我们手动挂上代理。
  • 规则（Rule）：所有请求根据配置文件规则进行分流
  • 直连（ Direct ）：所有请求直接发往目的地

各浏览器的proxy代理配置

​ 如果不是选择使用全局代理配置，我们浏览器访问外网需要手动代理配置，需要用到各浏览器的porxy插件，然后代理配置我们本地的clash代理的端口。

一、火狐

​ 火狐的FoxyProxy代理直接在火狐的扩展主题商店下载就行。

FoxyProxy下载

​ 火狐浏览器的右上角三条杠点击展开，然后点击设置，进入到火狐的设置界面；左下角扩展和主题，然后搜索FoxyProxy。

​ 搜索会出现FoxyProxy Standard，点击进去然后添加到Firefox。

FoxyProxy配置

​ 下载完后，FoxyProxy右上角的扩展会多出一个FoxyProxy。

​ 上方选择Proxies，添加代理配置。

  • ​ “标题或描述” : 就是设置代理的名字
  • ​ “Hostname” : 配置连接的地址，本地的代理就是127.0.0.1，如果是局域网内部其他的主机就填其他主机的ip
  • ​ “端口” : 代理开启的端口(clash默认是7890)

​

​ 接着浏览器右上角的扩展选择配置的代理就可以访问外网了。

二、Chrome

SwitchyOmega插件下载

http://crx4.com/18223.html

​

SwitchyOmega配置

​ 下载完后放到一个固定的文件夹位置，不能进行移动。下载完后会得到rar的包，将它进行解压，再将他拖拽到chrome浏览器里面去，在google的设置的左下角有个管理扩展程序，点击进入到扩展程序的配置，右上角开发者模式打开。

​ 然后将我们的文件拖拽进去.

​ 接着就会进入到SwitchyOmega的配置界面去，我们需要新建情景模式。

然后配置协议和ip、端口号,再点击应用选项就可以了

接着在右上角选择扩展点击proxy选择我们配置的模式，挂上代理就可以访问外网了。
